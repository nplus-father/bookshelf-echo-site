### [Lightning Memory-Mapped Database Manager (LMDB) 1.0](http://www.lmdb.tech/doc/)

- Lightning Memory-Mapped Database Manager (LMDB) 1.0 是一個基於 Btree 的資料庫管理函式庫，其設計靈感來自 BerkeleyDB API，但更為簡化。它將整個資料庫暴露在記憶體映射中，提供極高的效能、記憶體效率和完整的 ACID 事務語義，且無需頁面快取層或定期維護。LMDB 支援多執行緒和多程序併發讀寫，並透過寫時複製和多版本控制確保資料完整性，同時避免讀寫鎖定。
- Lightning Memory-Mapped Database Manager (LMDB) 1.0 is a Btree-based database management library, loosely modeled on the BerkeleyDB API but much simplified. It exposes the entire database in a memory map, offering extremely high performance, memory efficiency, and full ACID transactional semantics without requiring its own page caching layer or periodic maintenance. LMDB supports concurrent read/write access from multiple processes and threads, ensuring data integrity through copy-on-write and multi-versioning while avoiding reader-writer deadlocks.
