### [Accurate, Interdisciplinary and Transparent Structure-property Understanding with Deep Native Structural Reasoning](https://arxiv.org/abs/2607.07708v1)

- 研究人員推出了 SciReasoner，一個跨學科的科學基礎模型，能統一處理蛋白質、小分子和無機晶體的結構。該模型將複雜的空間、化學和拓撲資訊轉化為一種統一的「結構感知詞彙」，使其能夠進行推理。SciReasoner 不僅在 86 項基準測試中的 67 項上達到頂尖水平，顯著提升了蛋白質功能預測和化學合成規劃的準確性，更重要的是它能生成透明、可供檢驗的推理過程。這項進展將 AI 從「黑箱」預測工具轉變為可解釋的科學合作夥伴，對藥物發現和材料科學等領域意義重大。
- Researchers introduced SciReasoner, a multimodal foundation model for reasoning across proteins, small molecules, and inorganic crystals. The model works by discretizing complex structural information—coordinates, topologies, and periodicities—into a unified, structure-aware vocabulary that serves as evidence for its predictions. SciReasoner achieves state-of-the-art performance on 67 out of 86 scientific benchmarks, significantly improving tasks like protein function prediction and chemical retrosynthesis. Its key contribution is generating transparent reasoning traces that experts found comparable or superior to those from a frontier LLM, moving scientific AI from black-box prediction to an interpretable discovery tool.

### [Co-LMLM: Continuous-Query Limited Memory Language Models](https://arxiv.org/abs/2607.07707v1)

- 研究人員發表了 Co-LMLM，一種新型的「有限記憶語言模型」，它使用彈性的向量查詢從外部知識庫檢索事實，而非依賴僵化的結構化查詢。此方法讓知識庫能從任意文本中建立，突破了以往對維基百科等結構化資料的限制。其關鍵成果是，一個 3.6 億參數的小型 Co-LMLM 在事實準確性上媲美 gpt-4o-mini，並優於使用 40 倍數據量訓練的模型，展示了一條通往更高效、更真實且易於更新的 AI 系統之路。
- Researchers introduced Co-LMLM, a "Limited Memory Language Model" that retrieves facts from an external knowledge base using flexible vector queries instead of rigid, structured ones. This approach allows the knowledge base to be built from arbitrary text, moving beyond sources like Wikipedia. A small 360M parameter Co-LMLM achieved factual accuracy comparable to gpt-4o-mini and outperformed models trained on 40x more data, offering a promising path toward more efficient, factual, and easily updatable AI systems.
- score 4/5 · research · via arxiv `llm architecture` `retrieval-augmented generation` `factuality` `model efficiency` `knowledge base`

### [The Key to Going Linear: Analysis-Driven Transformer Linearization](https://arxiv.org/abs/2607.07706v1)

- 研究人員分析了為何將 Transformer 注意力機制線性化的現有方法常會降低模型品質，發現關鍵在於 softmax 依賴特定的正交投影。基於此洞見，他們引入了 sink token、短卷積和快取路由等結構性干預措施，以更準確地模擬原始注意力。此方法可在不重新訓練的情況下，將 LLaMA 等現有模型轉換為高效的長文本推理模型，其效能優於先前的後設（post-hoc）方法，並媲美複雜的自適應快取框架。
- Researchers analyzed why post-hoc linearization of Transformer attention often degrades quality, finding that softmax relies on key-dependent, rank-1 orthogonal projections. Based on this insight, they introduced structural interventions like sink tokens, short convolutions, and cache routing to better approximate the original mechanism. This approach converts pre-trained models like LLaMA for efficient long-context inference without retraining, outperforming prior post-hoc baselines and matching complex adaptive-caching frameworks on retrieval tasks.
- score 4/5 · research · via arxiv `transformers` `long-context` `inference optimization` `attention mechanism` `linear attention`

### [From Noisy Traces to Root Causes: Structural Trajectory Analysis and Causal Extraction for Agent Optimization](https://arxiv.org/abs/2607.07702v1)

- 研究人員開發了 STRACE 框架，用於優化基於大型語言模型的智慧體。該框架首先從大量執行軌跡中篩選出代表性的失敗模式，然後在單一軌跡內建立依賴關係圖，以精確定位導致失敗的根本原因。此方法能為 LLM 優化器提供高信噪比的上下文，避免了無關步驟或重複失敗的干擾。在一個困難的形式化驗證基準測試中，STRACE 成功將人類專家設計的智慧體成功率提升了 1.4 倍（從 42.5% 提高到 58.5%），展示了其在提升複雜智慧體性能方面的實用價值。
- Researchers developed STRACE, a framework for optimizing LLM-based agents by cleaning up noisy execution traces. The system first filters batches of traces to find representative failure patterns, then analyzes individual traces using a dependency graph to isolate the specific root-cause steps. This provides a high signal-to-noise context for the LLM optimizer, preventing it from being distracted by irrelevant actions or redundant failures. On a challenging formal verification benchmark, STRACE improved the success rate of a human-expert-designed agent by 1.4x (from 42.5% to 58.5%), demonstrating a practical way to make reflection-based agent improvement more effective.
- score 4/5 · research · via arxiv `agent optimization` `llm agents` `root cause analysis` `trajectory analysis` `reflection`

### [Breaking Database Lock-in: Agentic Regeneration of High Performance Storage Readers for Database Bypass](https://arxiv.org/abs/2607.07696v1)

- 研究人員開發了一個名為「Jailbreak」的系統，它利用 LLM 代理程式自動生成程式碼，以直接讀取專有的資料庫儲存檔案。透過分析資料庫的原始碼和文件，該代理程式能創建一個高效能的讀取器，繞過標準的資料庫引擎和驅動程式（如 JDBC/ODBC）。這種方法避免了傳統驅動程式在分析工作負載上的效能瓶頸，在 PostgreSQL 和 MySQL 上實現了高達 27 倍的吞吐量加速，並提供了一種通用方法來打破供應商鎖定，直接以 Apache Arrow 等高效格式存取數據。
- Researchers developed 'Jailbreak,' a system that uses an LLM agent to automatically generate code for reading proprietary database storage files directly. By analyzing a database's source code and documentation, the agent creates a high-performance reader that bypasses the standard database engine and drivers. This approach avoids the performance bottlenecks of traditional drivers for analytical workloads, achieving up to a 27x speedup on PostgreSQL and MySQL, and offers a generalizable method to break vendor lock-in by accessing data directly in efficient formats like Apache Arrow.
- score 4/5 · research · via arxiv `database` `llm agent` `data engineering` `performance` `vendor lock-in`

### [GLM 5.2 is nearly as accurate as a human book keeper](https://toot-books.pages.dev/blog/glm-5-2-vat-benchmark)

- 一項基準測試顯示，開源 AI 模型 GLM 5.2 能以近乎人類會計師的準確度，完成英國小型企業的季度增值稅 (VAT) 申報。該模型處理 59 筆交易的成本僅為 2.73 美元，最終稅額計算僅誤差 7 便士。儘管模型在一筆大額股本交易的法律分類上犯了嚴重錯誤，但此結果表明 AI 代理人已能以極低成本（相較於人工的 750-2100 英鎊）處理複雜的專業工作，預示著簿記等知識工作的自動化將快速到來。
- An open-weights model, GLM 5.2, prepared a quarterly UK VAT return with near-human accuracy in a benchmark test. The model processed 59 transactions for just $2.73, with the final VAT calculation being off by only 7 pence compared to a human-prepared return. While it made one serious legal classification error on a large capital transaction, the performance demonstrates that AI agents can now handle complex, real-world professional tasks like bookkeeping at a fraction of the human cost (£750-£2,100), signaling the rapid commoditization of such knowledge work.
