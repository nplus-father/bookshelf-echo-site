### [mienetic/mnema — 🧠 Mnema — Long-term memory for AI via MCP × Vector DB. Solves the context-window problem with pluggable vector backends (Chroma/Qdrant/sqlite-vec), hybrid search, decay scoring, and summarization.](https://github.com/mienetic/mnema)

- Mnema 是一個為 AI 代理設計的開源長期記憶層，旨在解決上下文視窗的限制。它透過可插拔的向量資料庫（如 Chroma、Qdrant）和模型上下文協定（MCP），與 Claude Desktop、Cursor 等客戶端整合。其重要性在於，它提供了一種實用方法，讓 AI 代理能跨會話維持持久且可搜尋的記憶，並利用混合搜尋與「遺忘曲線」等功能，在不超出上下文視窗或產生高昂 token 成本的情況下，有效回憶相關資訊。
- Mnema is an open-source long-term memory layer for AI agents, designed to solve the context window problem. It uses a pluggable vector database backend (Chroma, Qdrant, etc.) and integrates with clients like Claude Desktop and Cursor via the Model Context Protocol (MCP). This is significant because it provides a practical way for agents to maintain persistent, searchable memory across sessions, using hybrid search and a "forgetting curve" to recall relevant information without overloading the context window or incurring high token costs.

### [babarda/open-visualization-protocol — OVP: the open visualization protocol for describing, validating, and rendering information design identically, for humans, software, and AI agents, on any platform. 74 chart skills x 16 design languages.](https://github.com/babarda/open-visualization-protocol)

- 開源視覺化協定 (Open Visualization Protocol, OVP) 是一個為 AI 代理設計的新標準，旨在生成確定性、像素級一致的圖表。該協定包含 74 種圖表類型與 16 種設計語言，讓 AI 能透過精確指令而非模糊提示來渲染視覺內容。其重要性在於，它將不穩定的 AI 圖表生成過程轉變為可重現的工程任務，確保了自動化報告的視覺一致性，並透過內建規則強制執行誠實的數據呈現，解決了當前 AI 製圖的可靠性與一致性難題。
- The Open Visualization Protocol (OVP) is a new open standard for generating deterministic, pixel-identical charts, designed specifically for AI agents. It provides 74 chart types ("skills") and 16 design languages, allowing an agent to render a specific, validated visual from a precise command rather than a vague prompt. This matters because it replaces the unreliable process of asking LLMs for "a professional chart" with a reproducible, engineering-driven approach, ensuring visual consistency for automated reporting and enforcing honest data visualization practices.
