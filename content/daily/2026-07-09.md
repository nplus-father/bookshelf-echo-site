### [ELSA3D: Elastic Semantic Anchoring for Unified 3D Understanding and Generation](https://arxiv.org/abs/2607.06565v1)

- 研究人員發表了 ELSA3D，一個用於 3D 生成與理解的統一模型。它採用一種名為「彈性語義錨定」的新技術，將文字描述與 3D 物件表徵中特定的幾何尺度明確地連結起來。這種方法避免了將文字和 3D 數據扁平化為無差別序列的常見問題，透過在多個細節層次上建立稀疏而精確的語言與幾何連結，ELSA3D 在多項任務上取得了頂尖性能，同時顯著降低了約一半的計算成本。
- Researchers introduced ELSA3D, a unified model for both 3D generation and understanding. The model uses a novel technique called 'elastic semantic anchoring' to explicitly link text descriptions to specific geometric scales within a 3D object's representation, using a scale-aware octree tokenizer. This approach avoids collapsing text and 3D data into an undifferentiated sequence, enabling ELSA3D to achieve state-of-the-art results in text-to-3D, image-to-3D, and 3D captioning while roughly halving the computational cost and latency of a non-elastic version.

### [Hierarchical Acoustic-Semantic Modeling: Modality Separation and Semantic Coherence for Full-Duplex SLMs](https://arxiv.org/abs/2607.06540v1)

- 研究人員發現，現有的語音語言模型（SLM）將聲學與語意處理強制共享深層網絡參數，會導致優化信號衝突並降低效能。他們提出的新框架 Lychee-FD 採用分層架構，在深層分離這兩種模態，同時保持其對齊，在全雙工基準測試中將互動流暢度提升了 28.5%。此架構上的突破是邁向更自然、更智能對話式 AI 的重要一步，使其能處理即時、可中斷的對話，減少機械感。
- Researchers identified that forcing acoustic (speech) and semantic (meaning) processing to share deep network parameters in Spoken Language Models (SLMs) creates conflicting optimization signals, degrading performance. Their new framework, Lychee-FD, uses a hierarchical architecture to separate these modalities in deeper layers while maintaining alignment, achieving a 28.5% improvement in interaction fluidity on a full-duplex benchmark. This architectural insight is a significant step toward more natural and intelligent conversational AI that can handle real-time, interruptible dialogue without feeling robotic.
- score 4/5 · research · via arxiv `spoken language model` `conversational ai` `multimodal ai` `model architecture` `full-duplex`

It provides self-healing connections to prevent silent failures, runtime security features like "rug-pull" and poisoning detection to block malicious tool changes, and a compliance-mapped audit trail for all tool interactions. - [Graph Convolutional Attention: A Spectral Perspective on Graph Denoising and Diffusion](https://arxiv.org/abs/2607.06546v1) — Researchers introduced Graph Convolutional Attention (GCA), a new mechanism that improves graph denoising, a core operation in graph diffusion models. The work demonstrates that standard linear attention is suboptimal because it learns a single average spectral filter, which fails on datasets with diverse graphs. GCA overcomes this by using graph-filtered queries and keys to implement an adaptive spectral denoising filter tailored to each input graph. This approach improves performance and can match complex graph transformers without expensive feature computations, enabling more efficient and effective graph generation. `arxiv`

This approach aims to build foundation models that are not only linguistically accurate but also culturally meaningful, ensuring equitable performance for low-resource languages and preserving cultural heritage. `arxiv`

Is evaluation possible in other species?](https://arxiv.org/abs/2607.06542v1) — Researchers have shown it is feasible to evaluate unsupervised dependency parsing of non-human primate communication, even without a gold-standard dataset for verification. They found that the statistical properties of primate communication sequences, specifically the rapid decay of their length distribution, ensure that a parser will retrieve a high proportion of correct grammatical links. This counterintuitive finding, which does not hold for human language, provides a computationally rigorous method to analyze the syntax of animal communication, a task previously considered intractable. `arxiv`
