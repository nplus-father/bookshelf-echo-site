### [loop-js/loop.js — A loop engineering framework — state a Goal; Rounds run until a skeptical, read-only Verify agent settles it.](https://github.com/loop-js/loop.js)

- loop.js 是一個用於「循環工程」的 JavaScript 框架，旨在透過 AI 代理自動化執行複雜任務。使用者定義一個「目標」(goal)，並由一個獨立的、持懷疑態度的「驗證」(verify) 代理來判斷任務是否完成。這種將執行與驗證分離的架構，可以防止工作代理過早地自行判斷成功，從而提高了自動化流程的可靠性。該框架還內建了成本、回合數等安全護欄，以及崩潰後恢復和任務排程功能，使其適用於需要穩定運行的長時間自動化任務。
- loop.js is a JavaScript framework for "loop engineering," designed to automate complex tasks with AI agents. Users define a goal, and a separate, skeptical "verify" agent determines when the task is truly done. This architecture provides robustness by separating execution from verification, preventing the worker agent from prematurely grading its own homework. The framework also includes built-in guards for cost and rounds, crash-safe state management, and scheduling, making it suitable for reliable, long-running autonomous tasks.
